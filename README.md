# Environmental Data Analysis & Statistical Modeling 🌍📈

## Project Overview
This project focuses on the quantitative analysis of Australia’s freshwater withdrawal data (1993–2022). It combines descriptive statistics with probability modeling to identify consumption patterns and evaluate resource risks.

## Key Objectives
* **Descriptive Analytics:** Identify trends, volatility, and outliers in environmental datasets.
* **Probability Modeling:** Use Normal and Binomial distributions to forecast resource availability.
* **Data Integrity:** Assess data skewness and variability (IQR, Variance) for sustainable decision-making.

## Technical Stack
* **Language:** Python
* **Libraries:** Pandas (Data Manipulation), SciPy (Statistical Distributions), Matplotlib/Seaborn (Visualization).
* **Tools:** Jupyter Notebook.

## Key Statistical Insights
* **Distribution Analysis:** Analyzed freshwater withdrawal with a mean of 10.5 BCM and a standard deviation of 2 BCM.
* **Probability Calculations:** Modeled specific scenarios (e.g., calculating the 34.6% probability of exceeding withdrawal thresholds over a 4-year period).
* **Variability:** Identified a significant range (13.52) and negative skewness in long-term consumption data, indicating periods of extreme resource pressure.

## Project Structure
* `/analysis_notebooks`: Jupyter Notebooks with Python implementation of statistical tests.
* `/data`: Historical datasets of annual freshwater withdrawals.
* `Statistical_Report.pdf`: Comprehensive summary of findings and methodology.
